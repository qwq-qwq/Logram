F.01 Method timing

Paired + and - fold
into a slow-method table.

For every entry Logram finds the matching exit on the same thread, respecting nesting — and sorts by duration. Timings are computed lazily: only when you enable them.

F.02 SQL / cust1

Live SQL
with parameters.

cust1 parameters substituted back into the query. Copy — and paste into psql.

SELECT u.id, u.name, r.code
FROM uba_user u
JOIN uba_role r ON r.id = u.role
WHERE u.disabled = 0
  AND u.name = 'sergey@perek.rest'
  AND r.code IN ('admin','ops')
