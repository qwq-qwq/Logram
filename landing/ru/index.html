02Возможности

Инструменты, которые
делают лог читаемым.

Каждая функция сделана ради одной вещи: быстрее увидеть то, что сломалось.

F.01 Method timing

Парные + и - собираются
в таблицу медленных методов.

Для каждой входящей операции Logram находит парный выход на том же потоке с учётом вложенности — и сортирует по длительности. Cmd+J — прыжок между парой.

runList1 842 ms
selectByID 648 ms
authByPwd 214 ms
insert 89 ms
getAppInfo 31 ms
F.02 SQL / cust1

Живой SQL
с параметрами.

Подстановка cust1-параметров обратно в запрос. Копируй — и вставляй в psql.

SELECT u.id, u.name, r.code
FROM uba_user u
JOIN uba_role r ON r.id = u.role
WHERE u.disabled = 0
  AND u.name = 'sergey@perek.rest'
  AND r.code IN ('admin','ops')
